# VKinder

Чат-бот на Python для подбора пользователей через VK API.

Проект выполнен как дипломная работа в рамках обучения Python-разработке.

---

## 📌 Что делает бот

Бот подбирает пользователей на основе данных профиля и отправляет результаты в чат.

Используемые параметры:
- возраст  
- город  
- пол  
- семейное положение  

Для каждого найденного пользователя:
- выбираются 3 самые популярные фотографии (по лайкам и комментариям)  
- отправляется ссылка на профиль  
- данные сохраняются в базе данных  

При повторном поиске уже показанные пользователи не дублируются.

## 🧠 Как работает приложение

Проект разделён на несколько логических частей.

**Точка входа (launch)**  
Запрашивает у пользователя токен и параметры поиска, после чего запускает основной процесс.

**Интерфейс**  
Отвечает за взаимодействие с пользователем: принимает входные данные и выводит результат.

**Логика бота**  
Обрабатывает параметры поиска, управляет процессом подбора пользователей и координирует работу всех компонентов.

**Работа с VK API**  
Отправляет запросы к API ВКонтакте для поиска пользователей и получения фотографий. Ответы приходят в формате JSON и обрабатываются внутри приложения.

**Работа с базой данных**  
Используется PostgreSQL:
- сохраняются найденные пользователи  
- проверяется, показывался ли пользователь ранее  
- предотвращаются повторы  

---

## 🔄 Поток работы

1. Пользователь вводит параметры поиска  
2. Бот отправляет запрос к VK API  
3. Получает список подходящих пользователей  
4. Для каждого пользователя:
   - получает список фотографий  
   - определяет самые популярные  
5. Отправляет результат пользователю  
6. Сохраняет пользователя в базе данных
